What kinds of operators are these? What do they do?

&&, ||, !

They're called logical operators.

&&: evaluates if both are true (color===red && shape===circle)

||: evaluates if either one is true (age >= 17 || withGuardian===true)

! : used to evaluate the negative of a condition.

!color===red will return true for color===blue, color===green, etc.

## If, Else, Else If **if** is used to evaluate if a condition is true. If it is then the code inside it will be executed.
**else if** is used to add another conditional to be evaluated for true. If it is true then the code inside will be executed.

**switch** evaluates if one or more multiple conditions are true. If so, the code inside the block will be executed.

## Loop-de-loop

For Loop

```javascript for (i=0; i<6; i++) { changeText("This is time number " + i) } ```
## For Loop * initialization: the counter variable is initialized at this value. ```javascript for (i=0; i<6; i++) { changeText("This is time number " + i) } ```
## For Loop * condition: the for-loop keeps running until this becomes false. ```javascript for (i=0; i<6; i++) { changeText("This is time number " + i); } ```

A Note on Infinite Loops

  • They occur in loops when the condition is never false.
  • They lead to the code running and running until your browser runs out of memory, which will break your script.
  • They are very bad. Avoid them by checking your conditions carefully.
## For Loop * update: After each loop, the counter variable is changed in this way. * "++" means incremented by 1. For example, if i was 2, it becomes 3. ```javascript for (i=0; i<6; i++) { changeText("This is time number " + i) } ```

While Loop

```javascript var i = 0; while (i<6) { changeText("This is time number " + i) i++; } ```
## While Loop * Great way to keep executing a block of code until something changes. * Make sure that the condition can eventually change, though * Increase the counter variable within the block (i++)

Do-While Loop

```javascript var i = 0; do { changeText("This is time number " + i) i++; } while (i<6) ```
## Do-While Loop * Doesn't seem too different, huh? * The main difference is that a do-while loop always executes the block at least once.

If Statement

```javascript if (totalPoints > 99) { congratulateWinner(); } ```

If-Else Statement

```javascript if (gradeLevel === "Senior") { sayAlmostDone(); } else { sayStayInSchool(); } ```
## If-Else Statement * Evaluates (checks) a condition * If the condition is true, executes the next block of code * Executes a different block if the condition is false

Switch

```javascript var color = "green"; switch (color) { case "green": speed = 35; break; case "yellow": speed = 45; break; case "red": speed = 0; break; default: speed = 10; } ```
## Switch * Allows for multiple options in cleaner way than if-else-else-else-... * Condition would normally not be defined right above the switch (that's not too imaginative)
## What about break; and default: ? * break; stops execution of the code in the blocks * The last case (or default, if it's last) do not need to end with break; * default: is used when the condition does not match the switch statement cases
